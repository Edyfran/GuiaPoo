3.1 O que são Variáveis?

Variáveis são um dos conceitos mais fundamentais em programação. Para entender de forma simples, pense em variáveis como caixas etiquetadas onde você pode guardar diferentes tipos de informações durante a execução do seu programa.

Analogia com o mundo real

Imagine que você está organizando uma festa e precisa armazenar diferentes itens. Você poderia usar várias caixas e etiquetar cada uma: uma caixa chamada "Refrigerantes" para guardar bebidas, outra chamada "Decorações" para os enfeites, e assim por diante.

Em programação, as variáveis funcionam exatamente assim. Você cria "caixas" com nomes específicos (como idade, nome ou total) e guarda valores dentro delas. Depois, sempre que precisar desses valores, você pode "abrir a caixa" referindo-se ao nome da variável.

Por que usamos variáveis?

Antes de aprender a criar variáveis, vamos entender por que elas são tão importantes:

  • Armazenar dados temporariamente - Permitem guardar informações durante a execução do programa
  • Reutilizar valores - Em vez de repetir o mesmo valor várias vezes, você o armazena e usa quando necessário
  • Tornar o código mais legível - Nomes significativos ajudam a entender o propósito dos dados
  • Manipular e transformar dados - Você pode atualizar os valores conforme necessário

Criando variáveis em Python

Em Python, criar uma variável é extremamente simples. Você não precisa declarar explicitamente o tipo de dados como em outras linguagens. Basta:

  1. Escolher um nome para sua variável
  2. Usar o operador de atribuição = (sinal de igual)
  3. Especificar o valor que deseja armazenar
Python
# Criando uma variável para armazenar a idade de uma pessoa
idade = 25

# Criando uma variável para armazenar um nome
nome = "Ana"

# Exibindo os valores das variáveis
print("A pessoa se chama", nome)
print("Ela tem", idade, "anos de idade")
A pessoa se chama Ana Ela tem 25 anos de idade

No exemplo acima:

  • Criamos uma variável chamada idade e armazenamos o número 25
  • Criamos outra variável chamada nome e armazenamos o texto "Ana"
  • Usamos a função print() para exibir os valores dessas variáveis
Entendendo a atribuição

O símbolo = em Python é chamado de operador de atribuição. Ele não significa "igual a" como na matemática, mas sim "recebe o valor de" ou "armazena o valor".

Quando você escreve idade = 25, está dizendo ao Python: "Crie uma variável chamada 'idade' e armazene o valor 25 nela".

Regras para nomes de variáveis

Nem todos os nomes são permitidos para variáveis em Python. Existem algumas regras importantes que você deve seguir:

Regras obrigatórias
  • Os nomes devem começar com uma letra (a-z, A-Z) ou underscore (_)
  • O restante do nome pode conter letras, números e underscores
  • Nomes de variáveis não podem começar com números
  • Nomes de variáveis não podem ser palavras reservadas do Python (como if, for, while, etc.)
  • Python diferencia maiúsculas de minúsculas - idade, Idade e IDADE são três variáveis diferentes
Python
# Exemplos de nomes VÁLIDOS
idade = 25
nome_completo = "Ana Silva"
_contador = 0
total123 = 1000
minhaVariavel = "teste"

# Exemplos de nomes INVÁLIDOS (não execute este código)
# 1numero = 10  # Não pode começar com número
# meu-nome = "João"  # Não pode conter hífen
# if = 20  # Não pode ser palavra reservada

# Python diferencia maiúsculas e minúsculas
Valor = 10
valor = 20

# Essas são duas variáveis diferentes
print("Valor com V maiúsculo:", Valor)  
print("valor com v minúsculo:", valor)
Valor com V maiúsculo: 10 valor com v minúsculo: 20
Boas práticas para nomear variáveis
  1. Use nomes descritivos - Prefira idade_usuario em vez de x ou a
  2. Use snake_case - Em Python, a convenção é usar letras minúsculas com palavras separadas por underscores (Ex: preco_total, nome_do_cliente)
  3. Evite nomes muito longos - calculo_da_media_final_dos_alunos é difícil de digitar. Prefira algo mais conciso como media_final
  4. Seja consistente - Se você começa a usar um estilo, mantenha-o em todo o código

Usando variáveis

Depois de criar uma variável, você pode usá-la em diferentes partes do seu código. Vamos ver alguns exemplos:

Python
# Criando variáveis
base = 10
altura = 5

# Usando variáveis em cálculos
area = base * altura
perimetro = 2 * (base + altura)

# Exibindo os resultados
print("Dimensões do retângulo:", base, "x", altura)
print("Área:", area)
print("Perímetro:", perimetro)

# Usando variáveis em textos
mensagem = "A área do retângulo é " + str(area) + " unidades quadradas."
print(mensagem)
Dimensões do retângulo: 10 x 5 Área: 50 Perímetro: 30 A área do retângulo é 50 unidades quadradas.

Observe que:

  • Criamos variáveis para armazenar as dimensões do retângulo
  • Usamos essas variáveis em cálculos para obter a área e o perímetro
  • Usamos a função str() para converter o número em texto ao combiná-lo com outras strings

Atribuição dinâmica

Uma característica poderosa do Python é sua tipagem dinâmica. Isso significa que uma variável pode armazenar diferentes tipos de dados em diferentes momentos. Você pode reatribuir uma variável com um valor de tipo diferente a qualquer momento:

Python
# Criando uma variável com valor numérico
contador = 10
print("Valor inicial de contador:", contador)
print("Tipo inicial:", type(contador))  # Mostra o tipo da variável

# Mudando o valor e o tipo da variável
contador = "dez"
print("Novo valor de contador:", contador)
print("Novo tipo:", type(contador))  # Agora o tipo mudou

# Mudando novamente
contador = True
print("Valor final de contador:", contador)
print("Tipo final:", type(contador))  # Mudou novamente
Valor inicial de contador: 10 Tipo inicial: Novo valor de contador: dez Novo tipo: Valor final de contador: True Tipo final:

No exemplo acima, a variável contador primeiro armazena um número inteiro (int), depois uma string (str), e finalmente um valor booleano (bool). O Python permite isso, o que torna a linguagem muito flexível.

Atenção!

Embora o Python permita mudar o tipo de uma variável, é recomendável evitar isso quando possível. Manter o mesmo tipo de dados para uma variável torna o código mais previsível e facilita a detecção de erros.

Atribuição múltipla

Python oferece formas convenientes de atribuir valores a múltiplas variáveis em uma única linha. Isso é especialmente útil quando você precisa inicializar várias variáveis ao mesmo tempo.

Método 1: Mesmo valor para múltiplas variáveis

Python
# Atribuindo o valor 0 a três variáveis de uma vez
x = y = z = 0

print("x =", x)
print("y =", y)
print("z =", z)
x = 0 y = 0 z = 0

Método 2: Valores diferentes para múltiplas variáveis

Python
# Atribuindo valores diferentes a múltiplas variáveis
nome, idade, cidade = "Carlos", 30, "São Paulo"

print("Nome:", nome)
print("Idade:", idade)
print("Cidade:", cidade)

# Outro exemplo útil: trocando valores entre variáveis
a = 5
b = 10
print("Antes da troca: a =", a, "e b =", b)

# Em muitas linguagens, você precisaria de uma variável temporária
# Mas em Python, pode fazer assim:
a, b = b, a

print("Depois da troca: a =", a, "e b =", b)
Nome: Carlos Idade: 30 Cidade: São Paulo Antes da troca: a = 5 e b = 10 Depois da troca: a = 10 e b = 5

A atribuição múltipla é especialmente útil quando você trabalha com funções que retornam múltiplos valores ou quando precisa desempacotar valores de estruturas de dados como listas ou tuplas.

Variáveis: perguntas frequentes

As variáveis precisam ser declaradas antes de usá-las?

Em Python, não é necessário declarar uma variável antes de usá-la. A variável é criada no momento em que você atribui um valor a ela. Porém, tentar usar uma variável que nunca foi atribuída causará um erro. 

# Isso funcionará
nome = "Maria"
print(nome)  # Exibe "Maria"

# Isso causará um erro
print(sobrenome)  # NameError: name 'sobrenome' is not defined
Como saber o tipo de uma variável?

Use a função type() para descobrir o tipo de dados que uma variável contém: 

x = 10
y = "Hello"
z = 3.14

print(type(x))  # 
print(type(y))  # 
print(type(z))  #

Resumo

  • Variáveis são como "caixas etiquetadas" que armazenam dados na memória do computador
  • Para criar uma variável, use o formato: nome_da_variavel = valor
  • Nomes de variáveis devem seguir regras específicas:
    • Começar com letra ou underscore
    • Conter apenas letras, números e underscores
    • Não ser palavras reservadas
  • Python é case-sensitive, portanto nome e Nome são variáveis diferentes
  • Python tem tipagem dinâmica, permitindo que uma variável mude de tipo durante a execução
  • É possível atribuir valores a múltiplas variáveis em uma única linha
Dica final

Ao nomear variáveis, pense sempre na legibilidade do seu código. Um bom nome de variável deve descrever claramente o que ela armazena. Isso torna seu código mais fácil de entender, tanto para outras pessoas quanto para você mesmo no futuro!
