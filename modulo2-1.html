2.1 Seu Primeiro Código em Python

Se você está começando agora sua jornada em programação com Python, parabéns! Você escolheu uma das linguagens mais amigáveis para iniciantes. Neste módulo, você escreverá seu primeiro programa e dará os primeiros passos nesse mundo fascinante.

O que esperar

Aqui você aprenderá:

  • Como escrever seu primeiro programa Python
  • Como exibir mensagens na tela
  • Como armazenar informações em variáveis
  • Como interagir com o usuário para obter dados

O famoso "Olá, Mundo!"

Por tradição, o primeiro programa que se escreve ao aprender uma nova linguagem é o "Olá, Mundo!" - um programa simples que apenas exibe essa mensagem na tela.

Em Python, exibir texto na tela é incrivelmente simples. Você usa a função chamada print(), que mostra o conteúdo entre parênteses no console (a tela onde os resultados aparecem).

Python
print("Olá, Mundo!")
Olá, Mundo!

Vamos entender o que está acontecendo neste código:

  • print() é uma função integrada do Python que exibe informações na tela
  • Os parênteses () indicam que estamos chamando a função
  • O texto "Olá, Mundo!" está entre aspas duplas, indicando que é um texto literal (uma string)
  • As aspas não aparecem na saída, elas apenas delimitam onde o texto começa e termina
Por que começamos com 'Olá, Mundo!'?

Esta tradição começou em 1978 com o livro "The C Programming Language", escrito por Brian Kernighan e Dennis Ritchie. Desde então, tornou-se uma espécie de ritual de iniciação para programadores.

Começar com um programa simples como este permite focar na sintaxe básica da linguagem antes de avançar para conceitos mais complexos. É como aprender a dizer "olá" quando estudamos um novo idioma!

Imprimindo mais mensagens

Vamos experimentar exibir mais de uma mensagem:

Python
# Exibindo múltiplas mensagens
print("Olá, Mundo!")
print("Estou aprendendo Python!")
print("Programação é divertida!")
Olá, Mundo! Estou aprendendo Python! Programação é divertida!

Observe que:

  • Cada print() exibe o texto em uma nova linha
  • A linha que começa com # é um comentário - o Python ignora esta linha, ela serve apenas para explicar o código para quem o lê

Exibindo números e fazendo cálculos

A função print() não exibe apenas texto - ela pode exibir números e até mesmo resultados de cálculos:

Python
# Exibindo números
print(42)

# Exibindo o resultado de uma operação matemática
print(5 + 3)
print(10 * 2)
print(20 / 4)
42 8 20 5.0

Note que:

  • Quando imprimimos números, não usamos aspas
  • O Python pode executar operações matemáticas diretamente dentro do print()
  • Na divisão (20 / 4), o resultado inclui casas decimais (5.0), mesmo quando o resultado é um número inteiro

Combinando texto e números

E se você quiser exibir texto e números juntos? Existem várias maneiras de fazer isso:

Python
# Método 1: Usando vírgulas para separar os elementos
print("Eu tenho", 25, "anos")

# Método 2: Convertendo números para texto com str() e usando + para juntar
idade = 25
print("Eu tenho " + str(idade) + " anos")

# Método 3: Usando f-strings (disponível a partir do Python 3.6)
idade = 25
print(f"Eu tenho {idade} anos")
Eu tenho 25 anos Eu tenho 25 anos Eu tenho 25 anos
Dica!

Das três opções para combinar texto e números, as f-strings (Método 3) são geralmente consideradas a maneira mais elegante e legível. Elas permitem inserir valores de variáveis diretamente dentro de strings usando chaves {}.

Usando variáveis

Até agora, estamos inserindo valores diretamente no código. Mas geralmente, queremos armazenar valores para reutilizá-los. Para isso, utilizamos variáveis.

Python
# Criando variáveis para armazenar informações
nome = "Ana"
idade = 25
cidade = "São Paulo"

# Exibindo as informações usando as variáveis
print("Nome:", nome)
print("Idade:", idade)
print("Cidade:", cidade)

# Criando uma mensagem personalizada com as variáveis
mensagem = f"{nome} tem {idade} anos e mora em {cidade}."
print(mensagem)
Nome: Ana Idade: 25 Cidade: São Paulo Ana tem 25 anos e mora em São Paulo.

Neste exemplo:

  • Criamos três variáveis (nome, idade e cidade) e atribuímos valores a elas
  • Usamos estas variáveis nas chamadas para print()
  • Criamos uma variável mensagem que combina todas as informações em uma única frase
Analogia com o mundo real

Pense em variáveis como caixas etiquetadas onde você pode guardar diferentes tipos de valores. Cada caixa tem um nome (o nome da variável) e pode armazenar um valor. Quando você precisa usar esse valor, basta mencionar o nome da caixa!

Interação com o usuário

Um programa se torna mais interessante quando pode interagir com o usuário. Em Python, usamos a função input() para obter dados digitados pelo usuário:

Python
# Solicitando informações ao usuário
nome = input("Digite seu nome: ")
idade = input("Digite sua idade: ")

# Exibindo uma mensagem personalizada
print(f"Olá, {nome}! Você tem {idade} anos.")

# Calculando o ano de nascimento (aproximado)
# Precisamos converter a idade de string para número
ano_atual = 2024
ano_nascimento = ano_atual - int(idade)

print(f"Você nasceu aproximadamente em {ano_nascimento}.")
Digite seu nome: Maria Digite sua idade: 30 Olá, Maria! Você tem 30 anos. Você nasceu aproximadamente em 1994.

Vamos analisar este código:

  • A função input() exibe uma mensagem e espera que o usuário digite algo
  • O valor digitado pelo usuário é armazenado na variável (como nome ou idade)
  • Importante: O input() sempre retorna o que o usuário digita como uma string (texto)
  • Para fazer cálculos com números, precisamos converter a string para um valor numérico usando int() para números inteiros ou float() para números decimais
Atenção!

Se o usuário digitar algo que não pode ser convertido para o tipo esperado (por exemplo, digitar "vinte" quando esperamos um número para int(idade)), o programa dará erro. Mais adiante, você aprenderá como tratar esses casos com estruturas de tratamento de erros.

Aplicação Prática: Calculadora Simples

Vamos combinar o que aprendemos para criar algo útil: uma calculadora simples que solicita dois números ao usuário e exibe a soma:

Python
# Calculadora simples
print("=== Calculadora Simples ===")

# Obtendo os números do usuário
numero1 = input("Digite o primeiro número: ")
numero2 = input("Digite o segundo número: ")

# Convertendo as entradas para números
numero1 = float(numero1)
numero2 = float(numero2)

# Realizando as operações
soma = numero1 + numero2
subtracao = numero1 - numero2
multiplicacao = numero1 * numero2
divisao = numero1 / numero2

# Exibindo os resultados
print(f"\nResultados para {numero1} e {numero2}:")
print(f"Soma: {soma}")
print(f"Subtração: {subtracao}")
print(f"Multiplicação: {multiplicacao}")
print(f"Divisão: {divisao}")
=== Calculadora Simples === Digite o primeiro número: 10 Digite o segundo número: 5 Resultados para 10.0 e 5.0: Soma: 15.0 Subtração: 5.0 Multiplicação: 50.0 Divisão: 2.0

Esta calculadora simples demonstra vários conceitos que vimos:

  • Uso da função print() para exibir informações
  • Uso da função input() para obter dados do usuário
  • Conversão de strings para números com float()
  • Armazenamento de valores em variáveis
  • Realização de operações matemáticas
  • Formatação de saída com f-strings
Analogia com o mundo real

Programar é como dar instruções a um ajudante muito literal. Se você fosse explicar a alguém como fazer um sanduíche, precisaria listar cada passo: pegar o pão, abrir, colocar os ingredientes, etc.

Em Python, você está dando instruções ao computador da mesma forma - ele seguirá exatamente o que você escreveu, na ordem que você especificou. Isso significa que precisamos ser claros e precisos nas nossas instruções.

Resumo

  • Função print(): Exibe informações na tela
    • Pode exibir texto (entre aspas): print("Olá")
    • Pode exibir números: print(42)
    • Pode exibir resultados de cálculos: print(5 + 3)
    • Pode combinar texto e valores: print(f"Resultado: {5 + 3}")
  • Variáveis: Armazenam valores para uso posterior
    • Criadas com nome_variavel = valor
    • Podem armazenar texto, números, e outros tipos de dados
    • Podem ser usadas em cálculos ou exibidas com print()
  • Função input(): Obtém dados do usuário
    • Retorna sempre uma string (texto)
    • Para usar como número, é necessário converter: int() ou float()
  • Conversão de tipos:
    • str(): Converte para texto
    • int(): Converte para número inteiro
    • float(): Converte para número decimal
