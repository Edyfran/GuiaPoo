4.1 Estruturas Condicionais em Python

Em programação, frequentemente precisamos que nosso código tome decisões. Por exemplo, podemos querer executar uma ação apenas se um usuário tiver idade suficiente, ou mostrar diferentes mensagens com base na temperatura. Para isso, usamos estruturas condicionais.

O que aprenderemos

Nesta seção, você aprenderá:

  • Como usar a instrução if para executar código condicionalmente
  • Como adicionar alternativas com elif e else
  • Como trabalhar com operadores de comparação
  • Como combinar condições usando operadores lógicos
  • Como usar estruturas condicionais aninhadas

A Instrução if

A estrutura condicional mais básica em Python é a instrução if. Ela permite que você execute um bloco de código apenas se uma condição for verdadeira.

Python
# Estrutura básica do if
idade = 18

if idade >= 18:
    print("Você é maior de idade.")
    print("Você pode dirigir.")
Você é maior de idade. Você pode dirigir.

Vamos analisar este código:

  • A palavra-chave if inicia a estrutura condicional
  • idade >= 18 é a condição que está sendo verificada
  • O sinal de dois-pontos : indica o início do bloco condicional
  • As linhas indentadas abaixo do if formam o bloco de código que será executado se a condição for verdadeira
  • Se a condição for falsa, o bloco de código é simplesmente ignorado
Lembrete sobre indentação

Em Python, a indentação é crucial! Todo o código que faz parte do bloco if deve estar indentado (geralmente com 4 espaços). Quando você volta ao nível de indentação anterior, está indicando que o bloco terminou.

Adicionando Alternativas: else

E se quisermos executar um código diferente quando a condição for falsa? Para isso, usamos a instrução else.

Python
# Usando if com else
idade = 16

if idade >= 18:
    print("Você é maior de idade.")
    print("Você pode dirigir.")
else:
    print("Você é menor de idade.")
    print("Você não pode dirigir ainda.")
Você é menor de idade. Você não pode dirigir ainda.

Aqui, como idade é 16 (menor que 18), a condição idade >= 18 é falsa. Portanto, o bloco else é executado.

Analogia

Pense em estruturas condicionais como cruzamentos em uma estrada. O if é como uma placa que diz "Se você quiser ir para a cidade A, vire à direita". O else é como uma instrução complementar que diz "Caso contrário (se não quiser ir para a cidade A), vire à esquerda".

Múltiplas Condições: elif

Muitas vezes, precisamos verificar mais de duas possibilidades. Para isso, usamos a instrução elif (abreviação de "else if").

Python
# Usando if, elif e else
nota = 75

if nota >= 90:
    print("Conceito A")
elif nota >= 80:
    print("Conceito B")
elif nota >= 70:
    print("Conceito C")
elif nota >= 60:
    print("Conceito D")
else:
    print("Conceito F")
Conceito C

Este código verifica várias faixas de notas e imprime o conceito correspondente. As condições são verificadas em ordem, e assim que uma condição verdadeira é encontrada, o bloco associado é executado e as verificações subsequentes são ignoradas.

No exemplo, como nota é 75, a primeira condição (nota >= 90) é falsa, então passamos para a próxima. A segunda condição (nota >= 80) também é falsa. A terceira condição (nota >= 70) é verdadeira, então "Conceito C" é impresso e as verificações seguintes são ignoradas.

Importante!

A ordem das condições if/elif é importante! Python verifica as condições sequencialmente, de cima para baixo, e executa o primeiro bloco cuja condição for verdadeira.

Se você invertesse a ordem das condições no exemplo anterior, o resultado seria diferente, pois qualquer nota acima de 60 atenderia à primeira condição.

Operadores de Comparação

Para criar condições, usamos operadores de comparação, que comparam valores e retornam um resultado booleano (verdadeiro ou falso). Aqui estão os principais operadores de comparação em Python:

Operador Descrição Exemplo
== Igual a a == b (verifica se a é igual a b)
!= Diferente de a != b (verifica se a é diferente de b)
< Menor que a < b (verifica se a é menor que b)
> Maior que a > b (verifica se a é maior que b)
<= Menor ou igual a a <= b (verifica se a é menor ou igual a b)
>= Maior ou igual a a >= b (verifica se a é maior ou igual a b)
Python
# Exemplos de operadores de comparação
a = 10
b = 20

print("a == b:", a == b)  # Equal to
print("a != b:", a != b)  # Not equal to
print("a < b:", a < b)    # Less than
print("a > b:", a > b)    # Greater than
print("a <= b:", a <= b)  # Less than or equal to
print("a >= b:", a >= b)  # Greater than or equal to

# Comparando strings (comparação lexicográfica/alfabética)
nome1 = "Ana"
nome2 = "Carlos"
print("Ana vem antes de Carlos alfabeticamente:", nome1 < nome2)
a == b: False a != b: True a < b: True a > b: False a <= b: True a >= b: False Ana vem antes de Carlos alfabeticamente: True
Atenção!

Não confunda o operador de atribuição = com o operador de igualdade ==:

  • = é usado para atribuir um valor a uma variável
  • == é usado para verificar se dois valores são iguais

Usar = em uma condição pode causar erros ou comportamentos inesperados!

Operadores Lógicos

Muitas vezes, precisamos verificar várias condições simultaneamente. Para isso, usamos operadores lógicos:

  • and: retorna True se ambas as condições forem verdadeiras
  • or: retorna True se pelo menos uma das condições for verdadeira
  • not: inverte o resultado de uma condição (True vira False e vice-versa)
Python
# Exemplos de operadores lógicos
idade = 25
tem_carteira = True

# Usando o operador AND (ambas as condições devem ser verdadeiras)
if idade >= 18 and tem_carteira:
    print("Você pode dirigir.")
else:
    print("Você não pode dirigir.")

# Usando o operador OR (pelo menos uma condição deve ser verdadeira)
chovendo = True
tem_guarda_chuva = False

if chovendo or tem_guarda_chuva:
    print("Você não vai se molhar.")
else:
    print("Você vai se molhar.")

# Usando o operador NOT (inverte o resultado da condição)
esta_frio = False

if not esta_frio:
    print("Você pode sair sem casaco.")
Você pode dirigir. Você não vai se molhar. Você pode sair sem casaco.

Você também pode combinar múltiplos operadores lógicos em uma única expressão, usando parênteses para definir a precedência:

Python
# Combinando múltiplas condições
idade = 19
e_estudante = True
tem_carteira_estudante = False

# Verificando múltiplas condições
if idade <= 18 or (e_estudante and tem_carteira_estudante):
    print("Você tem direito à meia-entrada.")
else:
    print("Você deve pagar entrada inteira.")
Você deve pagar entrada inteira.
Tabelas de Verdade

Para entender melhor os operadores lógicos, é útil conhecer suas tabelas de verdade:

AND

  • True and True = True
  • True and False = False
  • False and True = False
  • False and False = False

OR

  • True or True = True
  • True or False = True
  • False or True = True
  • False or False = False

NOT

  • not True = False
  • not False = True

Estruturas Condicionais Aninhadas

Você pode colocar uma estrutura condicional dentro de outra. Essas são chamadas de estruturas condicionais aninhadas.

Python
# Estruturas condicionais aninhadas
idade = 17
acompanhado_por_adulto = True

if idade >= 18:
    print("Você pode assistir ao filme classificado para maiores.")
else:
    print("Você é menor de idade.")
    
    if acompanhado_por_adulto:
        print("Mas pode assistir ao filme porque está acompanhado por um adulto.")
    else:
        print("E não pode assistir ao filme porque não está acompanhado por um adulto.")
Você é menor de idade. Mas pode assistir ao filme porque está acompanhado por um adulto.

Embora seja possível aninhar muitos níveis de condicionais, isso pode tornar seu código difícil de ler e entender. Quando possível, prefira usar operadores lógicos para combinar condições em vez de usar muitos níveis de aninhamento.

Expressão Condicional (Operador Ternário)

Python oferece uma forma concisa de escrever condicionais simples em uma única linha, conhecida como expressão condicional ou operador ternário:

Python
# Expressão condicional (operador ternário)
idade = 20

# Forma extensa
if idade >= 18:
    status = "maior de idade"
else:
    status = "menor de idade"
print(f"Forma extensa: Você é {status}.")

# Forma concisa usando expressão condicional
status = "maior de idade" if idade >= 18 else "menor de idade"
print(f"Forma concisa: Você é {status}.")
Forma extensa: Você é maior de idade. Forma concisa: Você é maior de idade.

A sintaxe da expressão condicional é: 

valor_se_verdadeiro if condição else valor_se_falso

Essa forma é útil quando você quer atribuir um valor com base em uma condição simples, sem precisar escrever um bloco if-else completo.

Condições com Valor Truthy e Falsy

Em Python, qualquer expressão pode ser usada como uma condição, não apenas comparações. Python avalia automaticamente se um valor é considerado "verdadeiro" (truthy) ou "falso" (falsy).

Os seguintes valores são considerados falsos:

  • Constante False
  • Zero (0, 0.0)
  • String vazia ("")
  • Lista, tupla, conjunto ou dicionário vazios ([], (), set(), {})
  • None

Qualquer outro valor é considerado verdadeiro.

Python
# Exemplos de condições com valores truthy e falsy

# Lista vazia é considerada falsa
lista = []
if lista:
    print("A lista não está vazia.")
else:
    print("A lista está vazia.")

# String não vazia é considerada verdadeira
nome = "Python"
if nome:
    print(f"O nome é: {nome}")
else:
    print("Nenhum nome foi fornecido.")

# Verificando se uma variável tem um valor ou é None
resposta = None
if resposta:
    print(f"A resposta é: {resposta}")
else:
    print("Nenhuma resposta foi fornecida.")
A lista está vazia. O nome é: Python Nenhuma resposta foi fornecida.
Dica

Usar valores truthy e falsy pode tornar seu código mais conciso, mas também pode deixá-lo menos explícito. Para maior clareza, especialmente em código que será lido por outros (ou por você mesmo no futuro), considere usar comparações explícitas:

  • Em vez de if lista:, use if len(lista) > 0:
  • Em vez de if resposta:, use if resposta is not None:

Aplicações Práticas

Verificador de Elegibilidade para Empréstimo

Python
# Programa para verificar elegibilidade para empréstimo
print("=== Verificador de Elegibilidade para Empréstimo ===")

# Coletando informações do usuário
idade = int(input("Digite sua idade: "))
salario = float(input("Digite seu salário mensal: "))
tempo_emprego = float(input("Há quantos anos você está no emprego atual? "))

# Verificando elegibilidade
if idade < 18:
    print("Desculpe, você precisa ser maior de idade para solicitar um empréstimo.")
elif salario < 1000:
    print("Desculpe, o salário mínimo necessário é R$ 1.000,00.")
elif tempo_emprego < 1:
    print("Desculpe, você precisa estar no emprego atual por pelo menos 1 ano.")
else:
    # Pessoa é elegível, vamos calcular o valor máximo do empréstimo
    max_emprestimo = salario * 10
    
    if idade > 50:
        # Reduzir um pouco para pessoas mais velhas (maior risco)
        max_emprestimo *= 0.8
    
    if tempo_emprego > 5:
        # Aumentar para pessoas com estabilidade no emprego
        max_emprestimo *= 1.2
        
    print(f"Parabéns! Você é elegível para um empréstimo.")
    print(f"Valor máximo aprovado: R$ {max_emprestimo:.2f}")
=== Verificador de Elegibilidade para Empréstimo === Digite sua idade: 30 Digite seu salário mensal: 3000 Há quantos anos você está no emprego atual? 3 Parabéns! Você é elegível para um empréstimo. Valor máximo aprovado: R$ 30000.00

Calculadora de IMC com Interpretação

Python
# Calculadora de IMC (Índice de Massa Corporal)
print("=== Calculadora de IMC ===")

# Coletando informações do usuário
peso = float(input("Digite seu peso em kg: "))
altura = float(input("Digite sua altura em metros: "))

# Calculando o IMC
imc = peso / (altura ** 2)

# Interpretando o resultado
print(f"\nSeu IMC é: {imc:.1f}")

if imc < 18.5:
    print("Classificação: Abaixo do peso")
    print("Recomendação: Consulte um nutricionista para orientações sobre como ganhar peso de forma saudável.")
elif imc < 25:
    print("Classificação: Peso normal")
    print("Recomendação: Continue mantendo hábitos saudáveis!")
elif imc < 30:
    print("Classificação: Sobrepeso")
    print("Recomendação: Considere incluir mais atividade física na sua rotina.")
elif imc < 35:
    print("Classificação: Obesidade Grau I")
    print("Recomendação: Consulte um profissional de saúde para orientações específicas.")
elif imc < 40:
    print("Classificação: Obesidade Grau II")
    print("Recomendação: É importante buscar acompanhamento médico.")
else:
    print("Classificação: Obesidade Grau III")
    print("Recomendação: Busque ajuda médica para um plano de saúde personalizado.")
=== Calculadora de IMC === Digite seu peso em kg: 70 Digite sua altura em metros: 1.75 Seu IMC é: 22.9 Classificação: Peso normal Recomendação: Continue mantendo hábitos saudáveis!

Resumo

  • if: Executa um bloco de código se uma condição for verdadeira
  • else: Executa um bloco de código se a condição do if for falsa
  • elif: Permite verificar múltiplas condições em sequência
  • Operadores de comparação: ==, !=, <, >, <=, >=
  • Operadores lógicos: and, or, not
  • Expressão condicional: Forma concisa de escrever condicionais simples (valor_se_verdadeiro if condição else valor_se_falso)
  • Valores truthy/falsy: Python considera automaticamente se um valor é "verdadeiro" ou "falso" em um contexto condicional
Dica de Boas Práticas

Quando escrever estruturas condicionais:

  • Mantenha suas condições simples e legíveis
  • Use parênteses para agrupar condições complexas
  • Evite aninhar muitos níveis de if/else (geralmente mais de 3 níveis torna o código difícil de entender)
  • Considere refatorar lógicas complexas em funções separadas
