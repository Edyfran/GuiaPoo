5.2 Dicionários em Python

Dicionários são estruturas de dados extremamente versáteis em Python que armazenam pares de chave-valor. Diferente das listas que usam índices numéricos, dicionários permitem usar quase qualquer tipo de dado imutável como chave (strings, números, tuplas), dando acesso rápido aos valores associados.

O que aprenderemos

Nesta seção, você aprenderá:

  • Como criar e manipular dicionários
  • Como acessar, adicionar e remover elementos
  • Métodos comuns de dicionários
  • Como iterar sobre dicionários
  • Aplicações práticas de dicionários
Analogia

Pense em um dicionário Python como um dicionário de idiomas real ou uma agenda telefônica. Quando você quer encontrar o significado de uma palavra, você procura pela palavra (a chave) e obtém sua definição (o valor). Similarmente, em uma agenda telefônica, você procura pelo nome de uma pessoa (chave) para encontrar seu número de telefone (valor).

Criando Dicionários

Python
# Criando um dicionário com pares chave-valor
pessoa = {
    "nome": "Ana",
    "idade": 28,
    "profissão": "Engenheira",
    "cidade": "São Paulo"
}
print(pessoa)

# Dicionário vazio
vazio = {}
print(vazio)

# Dicionário com diferentes tipos de dados
misturado = {
    "string": "texto",
    10: "número como chave",
    2.5: "float como chave",
    (1, 2): "tupla como chave",
    True: "booleano como chave"
}
print(misturado)

# Criando com dict()
outro_dict = dict(nome="Carlos", idade=35, cidade="Rio de Janeiro")
print(outro_dict)
{'nome': 'Ana', 'idade': 28, 'profissão': 'Engenheira', 'cidade': 'São Paulo'} {} {'string': 'texto', 10: 'número como chave', 2.5: 'float como chave', (1, 2): 'tupla como chave', True: 'booleano como chave'} {'nome': 'Carlos', 'idade': 35, 'cidade': 'Rio de Janeiro'}
Importante

Apenas objetos imutáveis podem ser usados como chaves em dicionários. Isso significa que você pode usar strings, números e tuplas (desde que contenham apenas objetos imutáveis), mas não pode usar listas, conjuntos ou outros dicionários como chaves.

Acessando e Modificando Valores

Python
pessoa = {
    "nome": "Ana",
    "idade": 28,
    "profissão": "Engenheira",
    "cidade": "São Paulo"
}

# Acessando valores
print("Nome:", pessoa["nome"])
print("Idade:", pessoa["idade"])

# Modificando valores
pessoa["idade"] = 29
print("Idade atualizada:", pessoa["idade"])

# Adicionando novos pares chave-valor
pessoa["email"] = "ana@exemplo.com"
print("Dicionário atualizado:", pessoa)

# Removendo um par chave-valor
del pessoa["cidade"]
print("Após remover cidade:", pessoa)

# Verificando se uma chave existe
if "nome" in pessoa:
    print("A chave 'nome' existe no dicionário")
    
if "endereço" not in pessoa:
    print("A chave 'endereço' não existe no dicionário")

# Tentando acessar uma chave que não existe (causaria erro KeyError)
# print(pessoa["telefone"])  # KeyError: 'telefone'

# Método seguro para acessar: get() - retorna None ou um valor padrão se a chave não existir
print("Telefone:", pessoa.get("telefone"))  # Retorna None
print("Telefone:", pessoa.get("telefone", "Não informado"))  # Retorna "Não informado"

Métodos de Dicionários

Python
pessoa = {
    "nome": "Ana",
    "idade": 29,
    "profissão": "Engenheira",
    "email": "ana@exemplo.com"
}

# Obtendo todas as chaves
chaves = pessoa.keys()
print("Chaves:", list(chaves))

# Obtendo todos os valores
valores = pessoa.values()
print("Valores:", list(valores))

# Obtendo todos os pares chave-valor como tuplas
itens = pessoa.items()
print("Itens:", list(itens))

# Copiando um dicionário
pessoa_copia = pessoa.copy()
print("Cópia:", pessoa_copia)

# Limpando um dicionário
pessoa_copia.clear()
print("Após clear():", pessoa_copia)

# Atualizando o dicionário com outro
novos_dados = {"telefone": "123-456-789", "cidade": "São Paulo"}
pessoa.update(novos_dados)
print("Após update():", pessoa)

# Removendo e retornando um valor
email = pessoa.pop("email")
print(f"Email removido: {email}")
print("Após pop():", pessoa)

# Removendo e retornando o último par inserido (antes do Python 3.7 era aleatório)
ultimo_item = pessoa.popitem()
print(f"Último item removido: {ultimo_item}")
print("Após popitem():", pessoa)

# Obtendo valor, com um padrão se a chave não existir
site = pessoa.setdefault("site", "www.exemplo.com")
print("Site:", site)
print("Após setdefault():", pessoa)  # A chave foi adicionada com o valor padrão

Iterando sobre Dicionários

Python
pessoa = {
    "nome": "Ana",
    "idade": 29,
    "profissão": "Engenheira",
    "telefone": "123-456-789"
}

# Iterando sobre as chaves (padrão)
print("Iterando sobre as chaves:")
for chave in pessoa:
    print(chave)

# Iterando sobre as chaves (explícito)
print("\nIterando sobre as chaves explicitamente:")
for chave in pessoa.keys():
    print(chave)

# Iterando sobre os valores
print("\nIterando sobre os valores:")
for valor in pessoa.values():
    print(valor)

# Iterando sobre pares chave-valor
print("\nIterando sobre chave-valor:")
for chave, valor in pessoa.items():
    print(f"{chave}: {valor}")
Iterando sobre as chaves: nome idade profissão telefone Iterando sobre as chaves explicitamente: nome idade profissão telefone Iterando sobre os valores: Ana 29 Engenheira 123-456-789 Iterando sobre chave-valor: nome: Ana idade: 29 profissão: Engenheira telefone: 123-456-789

Dicionários Aninhados

Dicionários podem conter outros dicionários como valores, criando estruturas de dados complexas.

Python
# Dicionário aninhado
alunos = {
    "João": {
        "idade": 20,
        "curso": "Engenharia",
        "notas": [8.5, 7.0, 9.5]
    },
    "Maria": {
        "idade": 22,
        "curso": "Medicina",
        "notas": [9.0, 9.5, 8.0]
    },
    "Pedro": {
        "idade": 19,
        "curso": "Computação",
        "notas": [7.5, 8.0, 8.5]
    }
}

# Acessando informações aninhadas
print(f"Curso de João: {alunos['João']['curso']}")
print(f"Primeira nota de Maria: {alunos['Maria']['notas'][0]}")

# Calculando média das notas de um aluno
notas_pedro = alunos["Pedro"]["notas"]
media = sum(notas_pedro) / len(notas_pedro)
print(f"Média de Pedro: {media:.2f}")

# Adicionando uma nova informação em um dicionário aninhado
alunos["João"]["semestre"] = 4
print(f"Informações atualizadas de João: {alunos['João']}")

# Iterando sobre dicionários aninhados
print("\nInformações de todos os alunos:")
for nome, info in alunos.items():
    media = sum(info["notas"]) / len(info["notas"])
    print(f"{nome}: {info['curso']}, {info['idade']} anos, Média: {media:.2f}")
Curso de João: Engenharia Primeira nota de Maria: 9.0 Média de Pedro: 8.00 Informações atualizadas de João: {'idade': 20, 'curso': 'Engenharia', 'notas': [8.5, 7.0, 9.5], 'semestre': 4} Informações de todos os alunos: João: Engenharia, 20 anos, Média: 8.33 Maria: Medicina, 22 anos, Média: 8.83 Pedro: Computação, 19 anos, Média: 8.00

Dicionário por Compreensão

Semelhante às listas por compreensão, podemos criar dicionários de forma concisa usando compreensão de dicionário.

Python
# Criando um dicionário de quadrados de números
quadrados = {x: x**2 for x in range(1, 11)}
print("Quadrados:", quadrados)

# Criando um dicionário com números pares apenas
pares = {x: x**2 for x in range(1, 11) if x % 2 == 0}
print("Pares:", pares)

# Convertendo unidades de temperatura
celsius = {0, 10, 20, 30, 40, 50}
fahrenheit = {c: (c * 9/5) + 32 for c in celsius}
print("Temperatura em Fahrenheit:", fahrenheit)

# Invertendo um dicionário (troca chaves por valores)
original = {"a": 1, "b": 2, "c": 3}
invertido = {valor: chave for chave, valor in original.items()}
print("Original:", original)
print("Invertido:", invertido)
Quadrados: {1: 1, 2: 4, 3: 9, 4: 16, 5: 25, 6: 36, 7: 49, 8: 64, 9: 81, 10: 100} Pares: {2: 4, 4: 16, 6: 36, 8: 64, 10: 100} Temperatura em Fahrenheit: {0: 32.0, 10: 50.0, 20: 68.0, 30: 86.0, 40: 104.0, 50: 122.0} Original: {'a': 1, 'b': 2, 'c': 3} Invertido: {1: 'a', 2: 'b', 3: 'c'}

Aplicações Práticas de Dicionários

Python
# Exemplo 1: Contador de frequência de palavras
texto = """Python é uma linguagem de programação de alto nível, interpretada, 
de script, imperativa, orientada a objetos, funcional, de tipagem dinâmica e forte. 
Python é uma linguagem muito popular hoje em dia."""

# Removendo pontuação e convertendo para minúsculas
import string
texto = texto.lower()
for char in string.punctuation:
    texto = texto.replace(char, '')

palavras = texto.split()
contador = {}

# Contando as ocorrências de cada palavra
for palavra in palavras:
    if palavra in contador:
        contador[palavra] += 1
    else:
        contador[palavra] = 1

# Mostrando as palavras mais frequentes
print("Frequência de palavras:")
for palavra, freq in sorted(contador.items(), key=lambda x: x[1], reverse=True)[:5]:
    print(f"{palavra}: {freq}")

# Exemplo 2: Sistema de cadastro simples
print("\nSistema de Cadastro")
cadastro = {}

# Adicionando usuários
cadastro["user1"] = {
    "nome": "Carlos Silva",
    "email": "carlos@exemplo.com",
    "nivel_acesso": "admin"
}

cadastro["user2"] = {
    "nome": "Maria Souza",
    "email": "maria@exemplo.com",
    "nivel_acesso": "editor"
}

# Verificando acesso
def verificar_acesso(usuario, nivel_minimo):
    niveis = {"visitante": 0, "leitor": 1, "editor": 2, "admin": 3}
    
    if usuario not in cadastro:
        return "Usuário não encontrado"
    
    nivel_usuario = cadastro[usuario]["nivel_acesso"]
    if niveis[nivel_usuario] >= niveis[nivel_minimo]:
        return "Acesso permitido"
    else:
        return "Acesso negado"

print(f"Acesso de user1 como editor: {verificar_acesso('user1', 'editor')}")
print(f"Acesso de user2 como admin: {verificar_acesso('user2', 'admin')}")
Frequência de palavras: de: 7 python: 2 é: 2 linguagem: 2 uma: 2 Sistema de Cadastro Acesso de user1 como editor: Acesso permitido Acesso de user2 como admin: Acesso negado
Dicas para Trabalhar com Dicionários
  • Use o método get(): Sempre que não tiver certeza se uma chave existe, use dicionario.get(chave, valor_padrao) em vez de dicionario[chave] para evitar erros.
  • Valide a existência de chaves: Use if chave in dicionario: antes de acessar valores para evitar erros.
  • Considere defaultdict: Para dicionários com valores padrão, considere usar collections.defaultdict.
  • Chaves imutáveis: Lembre-se que apenas objetos imutáveis podem ser usados como chaves. Se precisar usar uma lista como chave, converta-a para uma tupla primeiro.
